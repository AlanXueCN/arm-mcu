Changes Made to FreeRTOS Source Distribution
--------------------------------------------

$Id$

1.  Downloaded FreeRTOS v7.0.1.

2.  Copied FreeRTOS/Source/* to lpc1768/FreeRTOS.

3.  Copied FreeRTOS/Source/include/* to lpc1768/FreeRTOS.

4.  Copied FreeRTOS/Source/portable/GCC/ARM_CM3/* to lpc1768/FreeRTOS.

5.  Copied FreeRTOS/Source/portable/MemMang/heap_3.c to lpc1768/FreeRTOS.

6.  Copied FreeRTOS/Demo/CORTEX_LPC1768_GCC_RedSuite/src/FreeRTOSConfig.h to
    lpc1768/FreeRTOS.

7.  Changed sprintf() to siprintf() in tasks.c to prevent unnecessarily
    pulling in floating point libraries at link time.

8.  Changed configUSE_TICK_HOOK to 0 in FreeRTOSConfig.h.

9.  Changed configGENERATE_RUN_TIME_STATS to 0 in FreeRTOSConfig.h.

10. Added missing #if ( configGENERATE_RUN_TIME_STATS == 1 ) around call to
    portCONFIGURE_TIMER_FOR_RUN_TIME_STATS() in tasks.c.

11. Added macros to map FreeRTOS interrupt handler names to framework names
    (xPortSysTickHandler to SysTick_Handler, xPortPendSVHandler to
    PendSV_Handler, and xPortSVCHandler to SVC_Handler).

12. Set INCLUDE_xTaskGetCurrentTaskHandle to 1 in FreeRTOSConfig.h.

===============================================================================

13. Copied lpc1768/FreeRTOS to str912faw44/FreeRTOS.

14. Copied port* from STR91x GCC contributed port to str912faw44/FreeRTOS.

15. Changed "LPC17xx.h" to "91x_lib.h" in FreeRTOSConfig.h

16. Changed configCPU_CLOCK_HZ to 48 MHz in FreeRTOSConfig.h.

17. Removed macro definitions for LPC17xx interrupt handler names (see #11
    above) from FreeRTOSConfig.h.

18. Added the following macro defintions to FreeRTOSConfig.h:

    #define configUSE_TIMER		TIM0
    #define configSysTickIRQPriority	5
    #define configCPU_PERIPH_HZ		48000000

19. Added #include <stdlib.h> to port.c (for abs()).

20. Removed definitions for macros portCONFIGURE_TIMER_FOR_RUN_TIME_STATS()
    and portGET_RUN_TIME_COUNTER_VALUE().

21. Defined configASSERT() as newlib assert() in FreeRTOSConfig.h.
